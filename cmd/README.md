### RSS news fetcher — компонент системы поиска новостей, который активно извлекает статьи из различных RSS-каналов, извлекает важную информацию и хранит её в векторной базе данных. ( Fetcher -  компонент или функция, отвечающая за получение данных или информации. Например, модуль, который получает данные с удалённого сервера. RSS (Really Simple Syndication) — это стандартизированный формат, используемый онлайн-издателями для распространения своего контента на другие сайты и сервисы. )
